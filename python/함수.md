##### parameter(매개변수)

##### argument({전달}인자, 인수)

함수를 호출할 때 함수에 잔달하는 입력 값



##### 기본 인자값

- 기본 값을 지정하여 함수 호출 시 인자값을 설정하지 않도록 함. 정의된 것보다 더 적은 개수의 인자들로 호출 가능

```python
def add(x, y=0):
	return x+y
	
# 값을 하나만 입력해도 실행된다.
```



##### 키워드 인자

- 직접 변수의 이름으로 특정 인자를 전달할 수 있다.
- 키워드 인자 다음에 위치 인자활용 X

```python
def add(x,y):
	return x+y
	
add(x=2, y=5)
add(x, y=5)
```



##### 가변 인자 리스트

- 함수가 임의의 개수 인자로 호출 될 수 있도록 한다
- 인자들은 **튜플**로 묶여 처리, 매개변수에 *을 붙여 표현
- 키값이 없다

```python
def add(*args):
	for arg in args:
	print(arg)
#*args는 바꿔도 되지만, 관례적으로 args로 사용한다	
add(2)
add(2,3,4,5)
```



##### 가변 키워드 인자

- 함수가 임의의 개수 인자를 키워드 인자로 호출될 수 있도록 한다
- **딕셔너리**로 묶여 처리, 매개변수에 **를 붙여 표현
  - ** 는 두번 묶는다는 뜻

```python
def family(**kwargs):
	for key, value in kwargs.items():
		print(key, ":" , value)
family(father='A', mother = 'B', ...)
```

# 함수 스코프



##### 스코프?

- 함수는 코드 내부에 지역 스코프를 생성
- 그외의 공간인 전역 스코프로 구분



##### 변수 수명 주기

- 빌트인 스코프 : 영원히
- 전역 스코프 : 모듈이 호출된 시점부터, 인터프리터가 끝날 때 까지
- 지역 스코프 : 함수가 호출될 때 생성, 종료될 때  까지



---

##### 이름 검색 규칙

- 파이썬에서 사용되는 이름들은 이름공간에 저장되어 있다.
- 아래와 같은 순서로 이름을 찾아나감
  - Local scope : 함수
  - Enclosed scope : 특정 함수의 상위 함수
  - Global scope : 함수 밖의 변수, Import 모듈
  - Built-in scope : 파이썬 안에 내장되어 있는 함수,속성

- 함수 내에서는 바깥 스코프의 변수에 접근 가능하나 수정은 안됨

---

##### global

- global 를 입력하면, 로컬에서 값을 사용 및 변경할 수 있다.

- global을 선언하기 전에 먼저 선언할 변수를 사용할 수 없다.
- 참조해서 변경할 경우만 사용하지만 좋은 방법은 아니다
- 매개변수, for루프 대상, 클래스 함수 정의 등으로 정의되지 않아야 한다.



#### nonlocal

- 전역을 제외하고 가장 가까운 스코프의 변수를 연결하도록 함
- global과는 달리 이미 존재하는 이름과 연결만 가능함

### 예외 처리

Try/except을 이용하여 예외처리를 할 수 있다. if문과 비슷!

try 실행

- 예외 O-> except 실행
- 예외 X-> except X

```python
try:
	#실행할 코드
except: #아무것도 안쓰면 모든 오류시에 except문 실행
	#오류시 실행할 코드
```

- **예외처리를 할때는 작은 범주부터 해주어야 한다.**

  



|         | 예외 o | 예외 X |
| ------- | ------ | ------ |
| try     | o      | o      |
| except  | o      |        |
| else    |        | o      |
| finally | o      | o      |



```python
except IndexError as err:
	print(f"{err}, 에러가 발생했습니다")
    # 에러값을 변수에 저장해서 사용 가능
```
