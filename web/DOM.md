### DOM

- html, xml 과 같은 문서를 다루기 위한 문서 프로그래밍 인터페이스
- 문서를 구조화 하고 구조화된 구성요소를 하나의 객체로 취급하여 다루는 논리적 트리 모델
- 문서가 구조화되어 있으며 각 요소는 객체로 취급
- 단순한 속성 접근, 메서드 활용 뿐만 아니라 프로그래밍 언어적 특성을 활용한 조작 가능
